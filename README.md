## :dart: Descripción ##

Este repositorio contiene la configuración base para iniciar un proyecto XR en Unity utilizando tecnologías como XR Plugin Management, OpenXR, y XR Interaction Toolkit (versión 3.0.1), así como perfiles de interacción con los Oculus Touch Controllers. El proyecto está diseñado para facilitar el desarrollo de aplicaciones de realidad extendida (XR), sirviendo como un "esqueleto" para proyectos VR, AR y MR.

Este repositorio es parte del curso "Desarrollo XR en Unity para Todos", ofrecido por la Universidad de los Andes como parte del proyecto Vivero Virtual (ViVi). 
El curso consta de 6 módulos, comenzando con la configuración de un proyecto XR y avanzando hacia la construcción de un Huerto VR. Los estudiantes utilizarán este repositorio como base a lo largo del curso, descargando módulos adicionales según avancen.

## :goggles: Vivero Virtual ##

Vivero Virtual (ViVi) es un Proyecto de Incubadora XR de la Universidad de los Andes que fue beneficiario de la beca Higher Ed XR Innovation Grant de Unity Social Impact y Meta Immersive Learning, gracias a sus propuestas de inclusión, impacto, viabilidad e innovación en la alfabetización del uso de las tecnologías, para aumentar el acceso a hardware AR/VR, el contenido educativo de alta calidad y otros recursos que ayudarán a los educadores a crear o mejorar programas XR innovadores.

ViVi busca llevar el conocimiento de la Realidad Extendida (realidad virtual, aumentada y mixta) a toda Iberoamérica. Para ello, ViVi propone diferentes líneas de acción como cursos virtuales masivos en Español, Campamentos para aprendizaje intensivo y préstamo de equipos de realidad virtual. Este proyecto busca generar una Comunidad de aprendizaje alrededor de las diferentes líneas de acción.

## :gear: Requisitos del Sistema ##
1. Unity Hub
2. Unity Editor 2021.3.16f1


## :rocket: Instalación ##

Para comenzar a trabajar con este proyecto en tu entorno local, sigue estos pasos:
### Para Windows y Mac:
1. Abre la consola de comandos (Command Prompt en Windows o Terminal en Mac).
2. Clona el repositorio en tu máquina local:
   ```git clone https://github.com/viverovirtualuniandes/proyectoXR-Base.git```
3. Abre Unity Hub.
4. Selecciona el botón "Add" para añadir un nuevo proyecto.
5. Navega a la carpeta que acabas de clonar y selecciónala (proyectoXR-Base).
6. Abre el proyecto.
7. Una vez que el proyecto haya cargado en Unity, sigue los pasos indicados en el curso para crear un proyecto XR.
